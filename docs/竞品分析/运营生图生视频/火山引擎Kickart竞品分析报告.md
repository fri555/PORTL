## 1. 产品概述

### 1.1 Kickart 是什么

**Kickart**（前身"创作Agent"）是字节跳动旗下火山引擎推出的一站式 **AI 营销创作平台**，专为电商和广告营销场景设计。其核心模式是 **"对话式视频生成"**——用户通过自然语言与 AI Agent 对话，即可完成从素材抓取、卖点提取、创意策划、分镜生成到视频合成的全流程。

**产品 slogan**：从商品链接到营销视频，一句话就够了。

### 1.2 发展历程

| 时间 | 里程碑 |
|------|--------|
| 2025年 | 创作 Agent 1.0/2.0 版本发布，实现基础的 AI 视频生成能力 |
| 2026年3月 | 创作 Agent 2.0 发布，集成 Seedance 2.0 模型 |
| **2026年6月17日** | **Kickart 3.0 正式发布**，全面升级为对话式视频生成模式 |

### 1.3 目标用户

- **电商商家**：淘宝/天猫、抖音电商、快手电商、小红书商家
- **品牌方**：需要批量制作营销素材的消费品品牌
- **广告代理商/MCN**：为多个客户高效产出投流素材
- **跨境卖家**：通过 BytePlus 国际版使用，支持多语种

### 2.2 五大核心功能

#### 2.2.1 一键成片（Video Agent）

| 步骤 | 功能 | 技术实现 |
|------|------|----------|
| Step 1 | 粘贴商品链接，自动抓取标题/描述/图片 | 多平台 URL 解析器 |
| Step 2 | AI 理解素材：卖点提取、主体识别、质量评估 | 自研营销 VLM 模型 |
| Step 3 | 生成 4 条差异化故事线（标题+风格+时长） | 创意 Agent 规划 |
| Step 4 | 逐镜分镜脚本 + 商品/人物/场景参考图 | 多模型协同调用 |
| Step 5 | 自动合成营销视频，支持下载/预览/发布 | Seedance 2.0 渲染 |

#### 2.2.2 爆款裂变（Viral Video Cloning）

- 上传参考爆款视频 → AI 多维拆解（风格、节奏、叙事逻辑、配音意图）
- 基于新商品信息**结构化复用**框架，非简单换皮
- 理解爆款**底层内容逻辑**，适配新 SKU/促销活动
- 支持 Seedance 2.0 模型，可选进阶/专业/快速三种质量模式

#### 2.2.3 对话式创作（Kickart 3.0 核心升级）

- **多轮自然语言对话**调整商品图、故事板、角色、场景
- 创作过程**可见可控**，非黑盒生成
- 支持局部修改（单镜重新生成、字幕擦除、文案/价格微调）
- **无需重做整个视频**即可针对性优化

#### 2.2.4 AI 特效模板

- 丰富的预设模板库（热点、节日、口播、UGC 风格、恶搞等）
- 支持 **480P / 720P / 1080P** 三种分辨率
- **支持模板定制**（case by case 评估）
- 支持 API 调用

#### 2.2.5 合规预审与内容分发

- 上线前自动生成**预审报告**（合规风险 + 视频质量 + 卖点清晰度 + 视觉表现）
- 覆盖抖音电商、巨量广告、抖音内容等审核场景
- 一键多平台分发到抖店、快手、小红书等

### 2.3 底层技术

| 技术组件 | 说明 |
|----------|------|
| **Seedance 2.0** | 字节自研 SOTA 视频生成大模型，支持 4K/60fps/15s 音视频联合生成 |
| **Seedance 2.0 mini** | 高性价比版本，成本降 50%，支持 1080p/10s |
| **营销 VLM** | 自研视觉语言模型，专为电商营销场景精调 |
| **多模态 Agent** | 自主规划分镜，调度 LLM、图像/视频/语音/音乐模型组合 |
| **时空注意力机制** | 动态权重分配，物理规律真实还原，多主体交互精度领先 |
| **双声道音频** | 背景音乐 + 环境音效 + 人声多轨并行，口型同步率 98% |

- **市场规模**：预计 2026 年底全球 AI 视频生成市场超过 **12 亿美元**
- **招银国际预测**：2027 年全球创意应用市场达 **546 亿美元**，AI 渗透率约 11%

## 4. 直接竞品深度对比

### 4.1 Kickart vs 即梦AI（字节内部竞品）

这是最特别的竞争关系——**同属字节生态，共享 Seedance 底层模型，但定位和路线截然不同**。

| 对比维度 | Kickart | 即梦AI |
|----------|---------|--------|
| **产品形态** | B2B SaaS 工作台 | C端 App/Web 创作工具 |
| **核心用户** | 企业/商家/代理商 | 个人创作者/博主 |
| **创作模式** | URL 输入 → Agent 全自动规划 | 手动 Prompt → 逐条生成 |
| **电商能力** | 🟢 内置（卖点提取/货架图/转化脚本） | 🟡 需自行组合 Prompt |
| **批量生产** | 🟢 爆款裂变/批量混剪 | 🟡 支持 CSV 参数化批量 |
| **合规审核** | 🟢 内置预审报告 | 🔴 无 |
| **分发能力** | 🟢 多平台一键发布 | 🟡 导出后手动分发 |
| **内容管理** | 🟢 云端资产管理 | 🟡 本地为主 |
| **API 集成** | 🟢 开放（SaaS/API/Skill） | 🟡 仅部分开放 |
| **定价** | ¥16,800/月起（企业级） | 约 ¥199/年（C端会员） |
| **月活用户** | N/A（B端产品） | 2037 万 MAU |
| **技术底座** | Seedance 2.0 + 营销 VLM | Seedance 2.0 |
| **多轮对话** | 🟢 Kickart 3.0 核心特性 | 🟡 基础支持 |

**关键洞察**：Kickart 和即梦 AI 本质上是同一底层能力的两种商业化路径——
- **即梦走"入口逻辑"**：抢占 C 端创作者心智，追求 MAU 规模，未来变现靠抖音生态；
- **Kickart 走"工具逻辑"**：服务 B 端企业，提供端到端专业工作流，直接收取 SaaS 费用。

字节内部认为"即梦的价值空间可能是剪映的十倍"（剪映 2024 年收入近百亿），但 Kickart 是当下直接创造企业级收入的产品。

## 9. 市场定位与差异化

### 9.1 Kickart 的差异化定位

```
                创作自由度
                    ↑
                    │    Runway/可灵
                    │    (专业创作工具)
                    │
                    │         Kickart ⭐
                    │         (电商营销全流程平台)
                    │
                    │    沃创
                    │    (快速铺货)
                    │              Canva
                    │              (通用设计)
                    │
                    └──────────────────────→ 电商场景深度
```

Kickart 的独特定位：**电商营销场景深度 × 创作自由度** 的交汇点。

### 9.2 核心差异化卖点

| # | 差异化能力 | 竞品覆盖情况 |
|---|-----------|-------------|
| 1 | **Agent 驱动的全流程自动化**（URL → 视频一键完成） | 仅沃创有类似能力，但不支持多轮对话调整 |
| 2 | **多轮对话式精细控制**（Kickart 3.0 核心） | 无竞品覆盖 |
| 3 | **爆款裂变的内容逻辑级复用**（非简单换皮） | 沃创/麦斯仅有模板级复刻 |
| 4 | **内置合规预审 + 质量评估** | 无竞品覆盖 |
| 5 | **SaaS + API + Skill 三种交付模式** | 竞品多为单一模式 |
| 6 | **Seedance 2.0 SOTA 模型 + 营销专用 VLM 精调** | 即梦同底层但无营销精调 |
| 7 | **多平台分发 + 云端资产管理** | 无竞品覆盖全流程 |

### 9.3 市场定位一句话

> **Kickart 不是"更好的 AI 视频生成工具"，而是"让电商企业拥有 AI 营销部门的操作系统"。**

## 1. 产品概述

### 1.1 知衣科技是什么

**知衣科技**（杭州知衣科技有限公司）成立于 2018 年，是一家以 **「AI + 大数据」** 为核心驱动力的服装行业 SaaS 领军企业。公司以「数据-设计-生产」闭环为核心理念，为服装企业提供从趋势洞察、智能选款、AI 设计到营销商拍的全链路数字化解决方案。

**公司使命**：用 AI 和大数据重新定义服装行业的选款与设计方式。

**Slogan**：AI 驱动全链路增长，助力服装企业降本增效。

### 1.2 公司发展历程

| 时间 | 里程碑 |
|------|--------|
| 2018 年 | 杭州知衣科技有限公司成立，创始团队来自北大、浙大、CMU 等院校 |
| 2019-2021 年 | 完成多轮融资（高瓴创投、君联资本、万物资本等），核心产品「知衣」SaaS 上线 |
| 2022 年 | 推出「炼丹炉」全域大数据分析平台，产品矩阵初具规模 |
| 2023 年 | FD（Fashion Diffusion）AI 设计产品内测，服务客户突破 5000 家 |
| 2024 年 | FD+ 正式上线，获全国「数据要素×」大赛三等奖；入选浙江省人工智能服务商 |
| 2025 年 | 连续登上央视报道、获国家级专精特新「小巨人」企业、七轮融资、服务近万家客户 |
| 2026 年 | 推出「知小衣 Agent」服装设计企划 AI 数字员工；FD+ 接入 Seedance 2.0；获电商 AI 挑战赛铜奖 |

### 1.3 目标用户

- **服装品牌方**：需要数据驱动企划与设计的品牌企业（如太平鸟、UR、波司登等）
- **ODM 厂商**：为品牌提供贴牌设计生产的供应商（需快速选款 + 生成设计提案）
- **电商卖家**：淘系/抖音/快手/跨境电商（需要爆款追踪 + 快速上新品）
- **设计师工作室**：利用 AI 提升设计效率的独立设计团队
- **出海品牌**：布局跨境电商的服装企业（通过「海外探款」产品）

#### 2.2.1 知衣 — 数据驱动的智能企划与选款平台

这是知衣科技的**核心旗舰产品**，也是公司同名品牌，定位为「服装行业 AI 大数据 SaaS」。

| **AI 选款引擎** | 覆盖 7000 万+ 商品库，支持面料/廓形/风格组合筛选 | 自研「柔性物体识别算法」+ 2000+ 服装标签体系 |
| **竞店监控** | 监控全网竞品动态，行业飙升榜单 | 50+ 电商平台数据采集（淘系/抖音/小红书/亚马逊等） |
| **爆款预测** | 爆款成功率预测模型 | ML 预测模型，款式录用率提升至 70%，爆款率翻倍 |
| **价格带分析** | 热销价格区间洞察，指导定价策略 | 实时 SKU 价格数据聚合 |

**核心指标**：
- 覆盖国内 **51%** 上市服装企业
- 电商大数据超 **1000 亿条**，小时级更新
- 款式录用率从传统 30% → **70%**
- 爆款率提升 **2 倍**

## 4. 直接竞品深度对比

### 4.1 知衣 vs Style3D（凌迪科技）— 最核心的国内竞对

Style3D（凌迪科技）是知衣科技在「AI + 服装数字化」赛道上最直接且实力最强的竞品。两者同为杭州企业、同样瞄准服装全链路，但技术路径和产品重心有本质差异。

| 对比维度 | 知衣科技 | Style3D（凌迪科技） |
|----------|---------|---------------------|
| **成立时间** | 2018 年 | 2015 年 |
| **核心技术** | AI 大模型 + 电商大数据 | 柔性仿真物理引擎 + 3D + AI |
| **核心产品** | 知衣（数据选款）、FD+（AI 设计/商拍） | Style3D Studio（3D 仿真）、DeepModa（AI Agent） |
| **数据侧** | 🟢 1000 亿+ 电商数据、50+ 平台 | 🟡 230 万+ 面料数据（偏设计前端） |
| **设计能力** | 🟢 FD+ AI 生图/改款/商拍 | 🟢 iCreate AI 灵感 + 3D 生成 |
| **3D 仿真** | 🔴 **缺失** | 🟢 全球领先（褶皱误差 <0.3mm） |
| **版型/工艺** | 🔴 **不涉及** | 🟢 AIGP 自动制版（研发中） |
| **面料数字化** | 🔴 **不涉及** | 🟢 Fabric 扫描仪（10 分钟 783 个参数） |
| **营销商拍** | 🟢 FD+ 深度覆盖 | 🟢 iWish 营销素材生成 |
| **AI Agent** | 🟡 知小衣（初期） | 🟢 DeepModa（7 大模块 40+ 环节） |
| **客户规模** | 近万家 | 2000+ 家 |
| **典型客户** | 太平鸟、UR、波司登、耐克、SHEIN | 安踏、波司登、李宁、SHEIN、Halara |
| **海外营收** | ~10% | ~50% |
| **融资** | 7 轮，准独角兽 | 多轮，估值 >10 亿美元 |
| **竞争壁垒** | 数据规模 + 电商生态深度 | 物理仿真引擎 + 3D 资产库 + 全链路 |

**关键洞察**：
- **知衣强在「数据」**：1000 亿级电商数据和 50+ 平台覆盖是 Style3D 无法短期追赶的护城河
- **Style3D 强在「3D 仿真」**：自研物理引擎和面料数字化能力使其在「设计→生产」环节有不可替代性
- **战略差异**：知衣走「数据驱动」（先看市场再设计），Style3D 走「技术驱动」（先建模再验证）。两者目标都是全链路，但切入点和路径不同
- **重叠加剧**：两者在「AI 设计 + 营销商拍」层面直接交火，客户也有大量重叠（波司登、UR 等同时使用两家产品）

### 4.2 知衣 vs WGSN — 趋势数据服务的国际对标

| **适用场景** | 电商选款、定价、备货 | 品牌年度企划、色彩/面料方向 |

| **价格** | 中等，多梯度配置 | 极高（~$25,000/年） |
| **客户类型** | 电商品牌/ODM/中小卖家 | 头部品牌/奢侈品/零售商 |

## 10. 战略建议

### 10.1 知衣科技的战略优先级评估

基于以上分析，知衣科技的当前战略优先级建议：

```
                     紧迫性
                        ↑
                        │  P0: 合规整改    P1: 3D能力补齐
                        │  P0: 设计精准度  P1: 国际化加速
                        │  P1: 开源防御    P2: Agent深化
                        │                 P2: 定价透明化
                        │
                        └────────────────────→ 重要性
```

### 10.2 具体建议

#### P0 — 立即行动

1. **合规整改（紧迫性最高）**
   - 6 月 1 日整改截止线：确保 APP 个人信息收集规范、禁止频繁自启动
   - 建立内部的 AI 数据合规体系，避免再次被通报
   - 对 B2B SaaS 而言，合规问题是客户信任的底线

2. **AI 设计精准度提升**
   - 服装结构理解（领型/袖口/接缝）是设计师抱怨最多的点
   - 引入版师反馈数据做 RLHF 微调，提升结构准确性
   - 设立「AI 出图 → 版师复核 → 反馈优化」的闭环流程

#### P1 — 重点项目

3. **3D 仿真能力补齐**
   - 自研或收购路径补齐 3D 能力，打通「设计→打版→生产」关键环节
   - 或与 CLO 3D / Style3D 建立更深度的集成合作
   - 这是与 Style3D 竞争的核心短板

4. **国际化加速**
   - 「海外探款」目前仍是早期产品，需要加大投入
   - 优先深耕日韩/东南亚市场（已有基础）
   - 建立英文/日文/韩文版本的产品矩阵

5. **开源防御策略**
   - 面对开源 AI 试衣模型的冲击，FD+ 需要从「能用」升级到「好用 + 可靠」
   - 强化企业级特性：品牌一致性、批量处理、工作流集成、权限管理
   - 开源模型很难替代「数据 + 设计 + 商拍」的一站式体验

#### P2 — 持续优化

6. **Agent 能力深化**
   - 「知小衣 Agent」是产品矩阵互联的关键，需要持续迭代
   - 目标：一个 Agent 驱动知衣选款 → FD+ 设计 → 炼丹炉验证 → 自动生成企划报告

7. **定价透明化**
   - 考虑推出标准化入门套餐 + 公开价格，降低中小企业决策门槛
   - 保持企业级的定制化高客单价方案

8. **客户教育**
   - 投入资源帮助客户理解 AI 工具的真实能力边界
   - 从「AI 替代设计师」转向「AI 增强设计师」的叙事

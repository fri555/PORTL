## 1. 产品概述

### 1.1 语雀是什么

**语雀**是蚂蚁集团旗下的结构化知识沉淀与协作平台。自研编辑器在中文排版、代码块、画板、Markdown 编辑方面是行业标杆。语雀的核心产品形态是"知识库→目录→文档"三层结构，是最符合中文用户使用习惯的知识组织方式之一。

**产品 Slogan**：结构化知识，体系化思考。

### 1.2 发展历程

| 时间 | 里程碑 |
|------|--------|
| 2018年 | 语雀正式发布 |
| 2020年 | 蚂蚁集团内部全面推广 |
| 2022年 | 支持私有化部署 |
| 2024年 | AI 搜索/问答功能上线 |
| 2026年 | 持续迭代，但AI功能迭代节奏低于行业平均 |

### 1.3 目标用户

- **研发/技术团队**：代码块、画板、UML 支持，研发友好
- **企业知识管理团队**：金融级数据安全
- **技术文档写作者**：Markdown+中文排版的标杆体验

### 1.4 公司概况

| 项目 | 内容 |
|------|------|
| **开发商** | 蚂蚁集团 |
| **部署方式** | SaaS + 私有化部署 |
| **AI 引擎** | 自研 |
| **核心优势** | 中文编辑器体验行业标杆、金融级安全、私有化部署 |

## 4. SWOT 分析

| 优势 | 劣势 |
|------|------|
| S1. 编辑器中文排版行业标杆 | W1. AI功能迭代节奏慢 |
| S2. 代码块/画板研发最友好 | W2. 生态集成能力弱（无IM/审批联动） |
| S3. 金融级数据安全 | W3. AI问答能力落后于钉钉/飞书 |
| S4. 支持私有化部署 | W4. 商业化保守，功能上线慢 |
| S5. 目录树清晰易用 | W5. 用户逐渐流向AI更强平台 |

| 机会 | 威胁 |
|------|------|
| O1. 政企私有化市场需求增长 | T1. 钉钉/飞书AI能力代际领先 |
| O2. 信创/等保合规刚需 | T2. 用户转向AI原生知识库 |
| O3. 编辑器+AI融合可能性 | T3. 开源方案侵蚀市场 |

## 二、缺失功能清单

### 2.1 文件上传与状态（PRD 3.2.4）

#### 1. 全流程状态机扩展
**当前**：仅3种状态 `已索引` / `解析中` / `待审核`，映射为 `processing | indexed | failed`
**PRD要求**：9种状态——`uploading / pending / processing / done / offline / upload_failed / process_failed / interrupted / cancelled` + 质量预警 `quality=low`。
**影响**：无操作态的文件上传过程中用户无法知道是卡在"上传中"还是"排队"还是"处理中"。

#### 2. 纯状态模式改造（去进度条）
**当前**：`uploadTasks` 含 `progress: number`，任务卡浮层展示进度条。
**PRD要求**：纯状态标识，无进度条。状态转换用标签切换动画代替。

#### 3. 上传中断与重试
**当前**：上传失败后无「重新上传」入口，文件条目消失。
**PRD要求**：
- 网络断开 → 保留文件条目，状态 `upload_failed`，提供「重新上传」
- <10s闪断 → 前端自动重试1次，用户无感知
- 「重新上传」直接调起文件选择器

#### 4. 上传中/待处理阶段取消
**当前**：无取消按钮。
**PRD要求**：`uploading`/`pending` 状态显示「取消」按钮 → 二次确认 → 状态变为 `cancelled` → 提供「重新处理」

#### 5. 文件去重与秒传
**当前**：同名文件直接共存，无校验。
**PRD要求**：前端 SHA256（分片流式）→ 命中秒传跳 `done` → 灰色「极速上传」标识 → 悬浮提示 → 「强制重新处理」入口

#### 6. 分片上传
**当前**：无。
**PRD要求**：5MB/块分片上传，断点续传，仅同会话内有效，OSS碎片7天清理。

#### 7. 文档版本管理
**当前**：无版本概念。
**PRD要求**：版本列表（时间线）/对比（并排diff）/回滚/备注/50版本上限自动归档。

#### 8. 相似文件处理
**当前**：无。
**PRD要求**：SimHash ≥95% → 弹窗选择：保留为新版本 / 覆盖旧版本 / 两者共存 / 取消上传。

### 2.2 文件搜索（PRD 3.2.6）

#### 9. 搜索自动补全
**当前**：搜索框为纯文本输入，无下拉建议。
**PRD要求**：300ms防抖，≤10条建议，前缀优先/中缀靠后，来源标注。

#### 10. 搜索结果高亮
**当前**：无关键词高亮。
**PRD要求**：`<mark>`黄色高亮匹配关键词，中文按词组，大文档前200字符。

#### 11. 拼写纠错
**当前**：无。
**PRD要求**：拼音/编辑距离检测 → 「您是否要找：{纠错建议}」。

#### 12. 热门搜索
**当前**：无。
**PRD要求**：首页搜索框下展示TOP10，不含敏感词。

#### 13. 相关搜索
**当前**：无。
**PRD要求**：搜索后「其他人还搜了」推荐。

---

### 2.3 标签系统（PRD 3.2.7）

#### 14. 标签选择器组件
**当前**：上传面板用toggle按钮组（预设6个标签），非下拉搜索。
**PRD要求**：下拉搜索 → 选中标签块展示+删除按钮。

#### 15. 上传时批量打标
**当前**：上传面板标签选择在主面板显示。
**PRD要求**：入口在「高级设置」折叠区，默认收起。

#### 16. 标签管理后台
**当前**：无管理员标签管理界面。
**PRD要求**：标签创建/编辑/删除/使用统计（管理员）。

---

### 2.4 知识库问答（PRD 3.3）

#### 17. 会话管理
**当前**：单会话，无会话列表/切换/新建/删除。
**PRD要求**：侧栏顶部会话下拉列表，新建/切换/删除/自动命名，保存20条历史会话。

---

### 2.5 权限管理（PRD 3.4）

#### 18. 物理删除二次确认
**当前**：回收站「彻底删除」直接执行，无确认。
**PRD要求**：二次确认弹窗，合规留存文档不删除。

#### 19. 密级标签UI
**当前**：无密级字段展示和筛选。
**PRD要求**：文件列表展示彩色密级标签（公共/内部/秘密/机密），支持按密级筛选。

---

### 2.6 审计与安全（PRD 3.4.4 / 3.5）

#### 20. 审计日志完善
**当前**：设置面板有审计记录 tab，内容为硬编码。
**PRD要求**：全链路记录，支持筛选和导出。

#### 21. PII检测与脱敏前端标识
**当前**：无。
**PRD要求**：文件详情/预览时展示脱敏标识。

### 2.7 其他

#### 22. Toast消息补全
**当前**：仅实现了删除相关Toast。

```
文件列表状态列（移除进度条，纯标识切换）：
┌───────────────┬───────────────┬──────────────────────┐
│ 文件名         │ 状态           │ 操作                   │
├───────────────┼───────────────┼──────────────────────┤
│ 方案v3         │ ✅ 已完成      │ [预览] [⋯]            │
│ 预算表          │ 🔄 处理中      │ —                     │
│ 合同扫描        │ ⏳ 排队中       │ [取消]                │
│ 旧方案          │ ❌ 上传失败     │ [重新上传]             │
│ 产品图          │ ✅ 已完成⚠️    │ [预览] [⋯] 悬浮提示质量  │
```
